# ESP32 Home Automation System

This project is an IoT based home automation system built using ESP32 and relay modules.

It allows users to control home appliances remotely through Arduino IoT Cloud as well as manual switches.

## Components Used

- ESP32 NodeMCU
- 4 Channel Relay Module
- Push Button Switches
- WiFi Network
- Arduino IoT Cloud

## Features

- Remote appliance control using IoT
- Manual switch control
- WiFi connection indicator
- Debounce protection for switches

## Working Principle

The ESP32 connects to WiFi and communicates with Arduino IoT Cloud.  
Users can control relays using a mobile dashboard or physical switches connected to the ESP32.

## Future Improvements

* Energy consumption monitoring system
* Advanced scheduling with AI-based automation
* Fault detection and alert system (overload, short circuit)
* Offline mode with local server control
